##  Tech Stack

- **Language:** Java  
- **Automation:** Selenium WebDriver  
- **Test Framework:** TestNG  
- **Behavior Driven:** Cucumber (BDD)  
- **Design Pattern:** Page Object Model (POM)  
- **Build Tool:** Maven  
- **Reporting:** TestNG reports

---

##  How to Run the Tests

1. Clone this repository:

2. Import the project into Eclipse or IntelliJ as a Maven project.

3. Ensure your **WebDriver executable** (chromedriver/geckodriver) is configured.

4. Run automation tests using:
- TestNG XML suite file
- Or right-click the test class and choose *Run as TestNG Test*
